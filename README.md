# Number Guessing Game

A simple Python command-line game where the user guesses a randomly generated number.

## Features
- Random number generation
- User input
- Too high / too low feedback
- Tracks attempts
- Handles invalid input using try/except

## Technologies Used
- Python
- Random module
- Git & GitHub

## What I Learned
This project helped me practise loops, conditionals, user input, error handling, and basic game logic in Python.
